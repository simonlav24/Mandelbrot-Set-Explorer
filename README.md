![](/Assets/Preview.png)

### Features:

* [X] explore and zoom by mouse
* [ ] control max iterations
* [X] render image in multiprocessing
* [ ] gui
* [X] coloring methods
  * [X] black and white
  * [X] colorful
  * [X] alternating black and white
  * [X] normal map
  * [X] image mapping
